# Even-Odd Sum Game

## Overview
The Even-Odd Sum Game is a simple interactive game where players must determine whether the sum of five randomly generated numbers is even or odd. Players are presented with two buttons to make their guesses.

## Project Structure
```
even-odd-sum-game
├── src
│   ├── index.html        # Main HTML document for the game
│   ├── styles
│   │   └── main.css      # Styles for the game
│   └── js
│       └── main.js       # JavaScript logic for the game
├── test
│   └── game.test.js      # Unit tests for the game logic
├── package.json           # npm configuration file
├── .gitignore             # Git ignore file
└── README.md              # Project documentation
```

## Getting Started

### Prerequisites
- Node.js and npm installed on your machine.

### Installation
1. Clone the repository:
   ```
   git clone <repository-url>
   ```
2. Navigate to the project directory:
   ```
   cd even-odd-sum-game
   ```
3. Install the dependencies:
   ```
   npm install
   ```

### Running the Game
1. Open `src/index.html` in your web browser to start the game.
2. Click the buttons to guess whether the sum of the five random numbers is even or odd.

### Running Tests
To run the unit tests, use the following command:
```
npm test
```
